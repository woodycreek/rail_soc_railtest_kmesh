# Install Instructions

1. clone this repo
2. open simplicity studio 5
3. clock fili/import
4. At the bottom of this dialog, clock more Import Options
5. Select General/Existing Project Into Workspae
6. Browse for the rail_soc_railtest_kmesh directory
7. Click Finish
